For my position as a Junior Software Engineer at Cherry, I had to solve this challenge:


"Build an application being able to retrieve the nearest pharmacy, given a geographical location as input, using the Public Milan Dataset"".


Of course, I admit that my level is beginner, and some points I could not achieve. I mention that: both servers use the same database and both run remotely without depending on any local server. In order to always be able to have a backup and keep track of the versions I use and the enhancements I used .git(both local and remote).


One extension I personally thought of for this app, which was tried but didn't find a happy ending, was to frame both solutions on a single page as well as displaying them on the same map in order to give the user an even more user-friendly experience.

For this presentation page I used Cherry's iconic colours, being a project that is made for them: color: #FC2438

Basic Solution

For the basic solution, an application must be implemented that can use coordinates (Latitude and Longitude) to find the nearest pharmacy to the user's address or an address entered by the user. In my case, I used this button below to be able to help the user to quickly detect his coordinates and locate himself on the map. In the link above you can access the application.

The application of the basic solutions was made using JavaScript ES6. I used MongoDB to be able to store all the data in a database. For deploying it remotely I used HEROKU & GitHub and the UI/UX was made using HTML, CSS & BOOSTRAP5.


BASIC SOLUTION KEYWORDS: Mathematical formula for determining the distance between two points.

(I recommend to open it with Google Chrome)

Advanced Solution

For the advanced solution, an application extends the basic solution, instead of entering the coordinates, the user enters the address around which they want to find the nearest pharmacy. In the link above you can access the application. For this application it was also intended to implement a way to tell the user how long it would take to travel to the nearest location (by car or walking). Unfortunately, this could not be achieved.
